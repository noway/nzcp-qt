# NZCP-Qt

An implementation of the NZCP Verifier App using Qt.

Uses [NZCP.c](https://github.com/noway/nzcp-c) library.

## Requirements
- `qt@6`
- `zbar`
- `imagemagick@7`

## Build
- `qmake`
- `./build_nzcp`
- `make`

## Run
- `open nzcp-qt.app`
